# Chapter 8. JavaScript Core Objects and Arrays

### Object Basics and Types
- Explains objects as collections of properties and methods, using real‑world examples like bank accounts to show how state and behavior are modeled in JavaScript.
- Introduces JavaScript object types: core objects (`Array`, `Date`, `String`, `Math`), HTML DOM objects, and browser objects used for page control and browser features.

### Core Objects and Creation
- Describes creating core objects with `new` (e.g., `new Date()`, `new String("Hello")`) and accessing members via dot notation (`obj.prop`) and method calls (`obj.method()`).
- Shows basic usage of `Date` for current time output and `String` for text storage and the `length` property.

### Arrays and the Array Object
- Defines arrays as structures storing multiple elements accessed by zero‑based indices, created via literals (`[]`) or `new Array()`.
- Covers dynamic sizing, the read‑only `length` property, storing mixed types, and key `Array` methods like `concat`, `join`, `reverse`, `slice`, `sort`, and `toString`.

### Date and String Usage
- Demonstrates `Date` methods such as `getFullYear()`, `getMonth()`, `getDate()`, and using time to change page background colors based on even/odd seconds.
- Explains `String` immutability and methods like `charAt`, `concat`, `indexOf`, `slice`, `substr`, `toUpperCase`, `replace`, `trim`, and `split`, including treating strings similarly to arrays via indexing.

# 8장. 자바스크립트 핵심 객체와 배열

### 객체 기본 개념과 종류
- 객체를 속성(프로퍼티)과 메서드로 구성된 단위로 소개하고, 은행 계좌 등 예시로 자바스크립트에서 상태와 동작을 함께 표현하는 방식을 설명함.
- 자바스크립트 객체 유형으로 코어 객체(`Array`, `Date`, `String`, `Math`), HTML DOM 객체, 브라우저 객체(BOM)를 구분해 용도와 범위를 정리함.

### 코어 객체 생성과 접근
- `new` 키워드로 `Date`, `String` 등의 코어 객체를 생성하고, 점 표기법으로 프로퍼티 접근 및 메서드 호출을 수행하는 기본 패턴을 다룸.
- `Date`로 현재 시간을 출력하고, `String`과 `length` 프로퍼티를 이용해 문자열 정보를 다루는 예제를 제시함.

### 배열과 Array 객체
- 배열을 0부터 시작하는 인덱스로 여러 값을 저장·접근하는 자료구조로 정의하고, `[]`와 `new Array()` 두 가지 생성 방식을 설명함.
- 배열 크기를 나타내는 `length`(읽기 전용), 서로 다른 타입 혼합 저장, `concat`, `join`, `reverse`, `slice`, `sort`, `toString` 등 주요 메서드 활용을 다룸.

### Date 객체와 String 객체 활용
- `getFullYear()`, `getMonth()`, `getDate()` 등 `Date` 메서드로 날짜·시간을 얻고, 초 단위에 따라 배경색을 바꾸는 실습 예제를 통해 활용 방법을 보여줌.
- 문자열 불변성, `charAt`, `concat`, `indexOf`, `slice`, `substr`, `toUpperCase`, `replace`, `trim`, `split` 등 `String` 메서드와 인덱싱을 통한 문자 접근 방식을 정리함.
